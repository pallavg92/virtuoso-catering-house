# Virtuoso Catering House — Brand Collateral

Digital-first brand assets matching the website's design system (charcoal `#2F352B`, ivory `#FDF8F0`, gold `#C9A768`, wine `#5E1B22`; Instrument Serif + Inter on the website, with close-match reliable substitutes — Cambria/Georgia + Calibri — used in Office documents so they render consistently on any recipient's machine).

## What's here

| Folder | Contents | Use |
|---|---|---|
| `business-cards/` | PNG business cards for Aarti Sharma, Pallav Goel, and a blank template, plus a shared back design | Upload directly to any print service, or use as a digital contact card / email attachment |
| `email-signatures/` | HTML signatures for Aarti, Pallav, and a blank template | Open in a browser, select the signature block, copy, paste into your email client's signature settings (Gmail: Settings → See all settings → Signature; Outlook: File → Options → Mail → Signatures) |
| `letterhead/` | `virtuoso-letterhead-template.docx` | Open in Word, type your letter starting where indicated — header/footer repeat automatically on every page |
| `invoice/` | `virtuoso-invoice-template.xlsx` | Fill in the blue cells (invoice #, dates, client info, line items) — amounts, subtotal, tax, and total calculate automatically |
| `company-profile/` | `virtuoso-company-profile.pptx` (7 slides) | Ready-to-send company profile covering About, Services, How We Work, Founders, Selected Work, and Contact. Export to PDF from PowerPoint (File → Export → Create PDF) for sharing |
| `pitch-deck/` | `virtuoso-pitch-deck-template.pptx` (8 slides) | Reusable proposal template — duplicate per client and fill in the bracketed `[placeholder]` sections (brief, approach, pricing, next steps) |
| `case-study/` | `virtuoso-case-study-template.docx` | One-pager template for writing up a real project once completed — duplicate per project and fill in the bracketed sections |

## Notes

- **Placeholder content**: the "Selected Work" slide in the company profile and any bracketed `[text]` in the pitch deck / case study templates are illustrative — replace with real project details as they become available. No client/brand names are used anywhere in this collateral.
- **Business card back** is shared across all fronts — use `business-card-back.png` with any of the front designs.
- **Fonts**: Office documents (docx/xlsx/pptx) use Cambria/Georgia + Calibri instead of the website's Instrument Serif/Inter, since those aren't standard Office fonts and would otherwise get silently substituted on a recipient's machine, risking layout shifts. The small italic "VCH" wordmark mark still uses Instrument Serif where it appears, since that's a short, fixed element unaffected by substitution.
- **Editing**: all files are original/editable (not flattened images), so text, colors, and layout can be adjusted directly in Word/Excel/PowerPoint.
